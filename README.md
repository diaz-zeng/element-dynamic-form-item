# element-dynamic-form-item

## 基于element-ui的动态表单项组件

### 安装

``` bash
yarn add element-dynamic-form-item
```

### 引用(打包还有点问题)

``` typescript
import { DynamicFormItem } from "element-dynamic-form-item/src/packages/dynamic-form-item";
Vue.use(DynamicFormItem);
```
